# Archive

Um [Archive](https://wiki.archlinux.org/title/Archiving_and_compression) zu entpacken:

    + lrzip
    + lzop
    + 7zip
    + unrar
    + zip
    + unzio

* `lrzip` für LRZ Formate
* `lzop`für LZO Formate
* `7zip` für 7z Archive
* `unrar` für rar Archive
* `zip` für zip Archive
* `unzip` für zip Archive
