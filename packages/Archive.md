# Archive

Um [Archive](https://wiki.archlinux.org/title/Archiving_and_compression) zu entpacken:

    + lrzip
    + lzop
    + 7zip
    + unrar
    + zip
    + unzio

* `lrzip` für LRZ Formate (.lrz)
* `lzop`für LZO Formate (.lzo, .tzo)
* `7zip` für 7z Archive (.7z)
* `unrar` für rar Archive (.rar)
* `zip` für zip Archive
* `unzip` für zip Archive (.zip)
